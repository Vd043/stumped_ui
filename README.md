## STUMPED UI

Flutter UI implementation of the three-screen cricket scoring flow:

- Landing screen with branding and **Initialize Match** call to action.
- **Set up your match** form screen.
- **Live Arena** scoring surface with action panel.

### How to run

1. Make sure you have Flutter installed (`flutter doctor`).
2. From the project root:

```bash
flutter pub get
flutter run
```

This project is configured to run on both Android and iOS (via Flutter). Integrate any business logic or backend as needed.
